# MrBinaer

## Install 

Linux

```console
wget https://github.com/mrtuxa/MrBinaer/releases/download/generic-linux-desktop-file/mrbinaer.desktop
sudo mv mrbinaer.desktop /usr/share/applications/
wget https://github.com/mrtuxa/MrBinaer/releases/download/v0.1.0-linux-86_64/mr_binaer
sudo mkdir -pv /usr/share/mrbinaer
sudo mv mr_binaer /usr/share/mrbinaer
wget https://github.com/mrtuxa/hack-ttf/raw/main/Hack-Regular.ttf
sudo mv Hack-Regular.ttf /usr/share/mrbinaer/font.ttf
```
